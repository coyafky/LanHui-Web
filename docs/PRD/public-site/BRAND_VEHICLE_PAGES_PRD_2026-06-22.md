## 1. 四个品牌页共同骨架

每个品牌页都必须具体回答：

1. 当前覆盖哪些车型。
2. 这些车型常见的升级需求是什么。
3. 蓝辉有哪些已经确认的改装件。
4. 哪些产品可以组合成一个完整方向。
5. 适配和施工有哪些边界。
6. 怎样联系门店确认。

统一模块：

```text
品牌车型 Hero
→ 车型选择
→ 该品牌车主常见需求
→ 四类轻改覆盖概览
→ 每个车型的推荐升级组合
→ 已确认改装件分类与产品卡
→ 代表项目图/细节图
→ 适配与施工说明
→ 相关产品知识与文章
→ 门店与企业微信咨询
```

产品参数和完整表格只放在后半段作为核对资料，不能作为页面主体。

## 2. 极氪品牌页 `/product/zeekr`

现有数据 23 款，可继续作为内容基础，但上线前需核实车型命名与实际产品。

### 2.2 页面主印象

`高端新能源座舱、空间与防护的轻改方案`

### 2.3 具体模块

#### Hero

- 极氪品牌车型范围。
- 强调座舱整体感、日常便利、尾箱空间和车辆防护。
- 使用极氪代表车型与已确认改装件组合图。

#### 车型导航

- 9X。
- 8X。
- 009。

#### 常见需求

- 座舱和内饰细节提升。
- 后排与尾箱空间整理。
- 底盘和车身易损区域保护。
- 上下车便利。

#### 四类轻改覆盖

| 项目 | 页面展示规则 |
|---|---|
| 电动踏板 | 有确认数据才展示车型款式 |
| 轮毂 | 作为后续内容位，未确认前不展示具体款式 |
| 底盘护板 | 展示底盘、门槛、后护板等防护方向 |
| 内饰改装 | 重点展示地板尾箱、座舱便利和保护件 |

#### 按车型方案

每个车型先展示 2—4 个推荐方向，例如：

- 9X：座舱便利 + 尾箱整理 + 防护配件。
- 8X：内饰保护 + 日常便利 + 底盘防护。
- 009：后排空间 + 地板总成 + 商务/家庭座舱。

不得仅显示“16 款、6 款、1 款”后直接进入商品网格。

#### 改装件分类

- 内饰便利。
- 地板与尾箱。
- 防护配件。
- 底盘/车身防护。
- 电动踏板（有数据时）。

#### 结尾

- 适配说明。
- 相关产品页。
- 相关极氪文章。
- `咨询极氪车型方案`。

## 6. 四品牌共同的产品卡

每张卡必须包括：

- 车型。
- 产品名称。
- 所属项目：电动踏板/轮毂/底盘护板/内饰改装/膜系。
- 解决的使用问题。
- 图片状态。
- 适配提示。
- `查看项目说明` 或 `咨询此款`。

不能只显示名称、序号和参数。
