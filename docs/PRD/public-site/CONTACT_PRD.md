## 1. 系统目标

让车主能够通过官网便捷地联系蓝辉——查看门店信息、拨打电话、添加微信或导航到店。咨询系统覆盖全站，不限于单个联系页。

系统必须保证：
- 电话和微信等联系方式真实有效，不使用假 400 或占位信息
- 咨询入口在 Header、产品页、文章页、门店详情页等关键位置可用
- 每次咨询行为可追溯埋点，但不采集个人电话号码和微信号
- 咨询渠道由后台 CMS 统一配置

## 4. 页面结构

### 4.1 `/contact` 联系聚合页

1. **Hero**：联系我们 + 背景图
2. **核心联系信息**：电话、微信、地址三列
3. **门店详情**：门店卡展示
4. **服务流程**：从需求到交付的流程说明
5. **品牌承诺**：品质保障、售后支持等

### 4.2 全站咨询入口

Header、产品详情、文章详情、门店详情、Footer 均有咨询入口。

## 5. 咨询渠道选择规则

```
门店有专属微信/电话 → 显示门店专属渠道
门店无专属渠道 → 显示总部企微
无企微 → 显示总部个微
无个微 → 显示总部电话
无电话 → 显示联系页
```

### 5.1 全站统一弹窗

- 支持企业微信和个人微信展示
- 场景化标题（根据来源页面传递上下文）
- 传入来源：product、article、store、contact、header、footer
- 有效二维码时正常显示，无效时提示"当前没有可用的咨询渠道"

### 5.2 弹窗状态管理

| 状态 | 展示 |
|---|---|
| 加载中 | 骨架屏 |
| 有效渠道 | 二维码 + 渠道说明 |
| 无渠道 | "当前没有可用的咨询渠道，请稍后再试" |
| 图片加载失败 | 渠道名 + "请截图保存" |
| 渠道停用 | "该渠道已停用" |
| 多次点击防抖 | 不重复创建弹窗实例 |

## 6. UI / 交互规范

### 6.1 视觉规范

- **弹窗**：`bg-zinc-900` 深色背景，`rounded-2xl`
- **电话 CTA**：`bg-orange-500` 实心按钮
- **微信入口**：`bg-emerald-500/10 text-emerald-400` 绿色调
- **二维码**：居中展示，适当内边距

### 6.2 弹窗交互

- 点击弹窗外部关闭
- ESC 键关闭
- 焦点锁定在弹窗内（Trap focus）
- 关闭按钮 `aria-label="关闭咨询窗口"`

## 7. 数据模型

### 7.1 当前实现（静态数据）

```ts
// src/lib/brand.ts
brand: {
  phone: '0757-22222222',      // 品牌总机
  wechat: 'fkycoya',           // 微信号
  qrCode: '/images/brand/wechat-qr.png',
  // ...
}
```

### 7.2 目标实现（DB 配置）

咨询渠道由后台 CMS 管理，存储于咨询渠道表：

```prisma
model ConsultationChannel {
  id          String   @id @default(cuid())
  type        String   // 'wechat' | 'enterprise_wechat' | 'phone'
  label       String   // 显示名称
  value       String   // 微信号/手机号
  qrCodePath  String?  // 二维码路径
  isActive    Boolean  @default(true)
  sortOrder   Int      @default(0)
  storeId     String?  // 可选，门店专属渠道
  createdAt   DateTime @default(now())
  updatedAt   DateTime @updatedAt
}
```

## 9. 当前实现差距

- 固定读取 `/images/brand/wechat-qr.png`（无后台配置）
- 固定显示"微信号：fkycoya（待补充）"
- `openWeChatModal()` 不能传来源、产品、文章或门店上下文
- Analytics 只允许通用 `click`，没有规划的 `contact_click`
- 没有咨询渠道数据库表、后台页面和公开解析 API
- 联系页仍有假 400 和 `qrCode: null` 占位
- 图片上传 API 当前只支持 `store`
- 渠道选择没有分级降级逻辑

## 10. 埋点

| 事件 | 触发条件 | 必需数据 |
|---|---|---|
| `contact_click` | 点击微信/电话/导航 | channel, sourceArea, entityType, entityId |
| `navigation_click` | 点击门店导航入口 | storeId |

禁止采集：微信号、手机号、聊天信息。
