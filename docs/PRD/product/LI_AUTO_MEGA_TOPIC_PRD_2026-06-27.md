## 0. 参考页面原则

本 PRD 参考蔚来 ES8 单车型页面和刚建立的理想 L9 PRD。MEGA 页面应继承同一套产品专题模式：

- Hero 展示车型定位、项目数量、场景锚点和咨询 CTA。
- 18 项目录以 HTML 卡片结构化呈现，不依赖长海报传达信息。
- 项目按真实 MPV / 家庭商务用车场景重组。
- 推荐组合用于销售沟通和用户决策，不写成强制套餐。
- 服务流程、FAQ、适配边界、SEO 和 JSON-LD 保持单车型专题统一规范。

不采用以下做法：

- 不把完整海报作为首屏主视觉。
- 不设计独立的「海报素材展示」模块。
- 不出现“理想官方授权”“原厂配件”“100% 无损安装”“不影响质保”等未经证明的承诺。

## 1. 背景

用户提供了一张理想 MEGA 轻改产品目录海报，标题为「热门轻改产品目录｜理想 MEGA 专属升级方案」。海报包含 18 项轻改产品，覆盖车身保护、玻璃隔热、外观个性、座舱地板、底盘防护、商务便利、灯光视觉和内饰养护。

理想 MEGA 与 L9/i8 不同，它更偏家庭 MPV、商务接待、多人出行和长途场景。页面内容需要突出大车身保护、上下车便利、二三排座舱易清洁、商务便利和外观细节升级。

一句话定位：

> 理想 MEGA 专题页是蓝辉面向理想 MEGA 车主的单车型轻改方案页，用于展示 18 项热门轻改产品，并按新车保护、商务座舱、外观个性、行车防护和灯光细节场景引导车主确认适配与施工。

## 2. 目标

### 2.1 用户目标

理想 MEGA 车主进入页面后，需要快速知道：

1. 我的理想 MEGA 可以做哪些热门轻改项目？
2. 哪些项目适合刚提车优先做？
3. 哪些项目适合 MPV 家庭/商务座舱、二三排使用和上下车便利？
4. 哪些项目偏外观个性、底盘防护、灯光视觉或内饰养护？
5. 哪些项目需要到店确认年份、配置、接口、安装位和施工风险？
6. 如何联系蓝辉确认方案和排期？

### 2.2 业务目标

蓝辉通过该页面实现：

1. 把「理想 MEGA 专属升级方案」从海报素材转成官网内容资产。
2. 承接 MEGA 车主对大空间 MPV 轻改、商务舒适和日常防护的搜索需求。
3. 给门店销售一条可直接转发的 MEGA 车型方案链接。
4. 通过项目点击、场景筛选和组合点击数据判断 MEGA 用户更关注保护、商务便利、外观还是防护。
5. 与理想 i8 / L9 一起形成理想品牌车型专题矩阵。

> 不同年份、批次、版本和配置的理想 MEGA 在尺寸、接口、安装位和结构上可能存在差异。页面项目只作为轻改方向参考，最终以到店确认和施工评估为准。

### 3.3 合规边界

页面不得出现以下未经证明的表达：

- 理想官方授权。
- 理想原厂配件。
- 官方同款。
- 不影响原车质保。
- 100% 无损安装。
- 永久质保。
- 全网最低。
- 性能提升、操控提升、制动提升、照明提升等不可验证承诺。
- 暗示与理想汽车存在品牌合作关系。

### 11.1 图片状态规则

| 状态 | 页面表现 | 使用条件 |
|---|---|---|
| `matched` | 展示真实项目图 | 图片已经确认适配该项目 |
| `generated-preview` | 展示 AI 预览图 + 「预览图」角标 | 后续为 MEGA 生成 4:3 产品图 |
| `pending-review` | 展示海报裁切或占位 + 「图片审核中」标签 | 当前来源图未拆分为最终产品图 |
| `missing` | 展示统一占位卡片 | 暂无可用素材 |

当前 PRD 默认 18 项图片状态为 `pending-review`。如后续生成或补齐项目图，建议统一规格为 `1448x1086`、`4/3`，与 ES8 素材规范一致。

### 11.2 筛选与排序规则

| 能力 | 一期要求 |
|---|---|
| 默认排序 | 按海报序号 01-18 |

### 11.3 移动端要求

- Hero 不超过一屏半。
- 项目卡片单列展示，触控区域足够大。
- 场景筛选可横向滚动或折叠，但不能遮挡正文。
- 图片容器固定比例，避免卡片跳动。
- CTA 在页面底部明确出现，不使用遮挡内容的浮层。

## 18. 风险与边界

| 风险 | 说明 | 处理 |
|---|---|---|
| 素材非最终施工图 | 当前只有用户海报，未拆成独立项目图 | 默认 `pending-review`，后续补真实图或生成预览图 |
| 车型适配差异 | MEGA 年份、版本、配置可能影响项目安装 | 所有涉及安装位项目加到店评估提示 |
| 电气/门体项目风险 | 主副驾电吸门涉及接口、门体结构和售后边界 | 必须到店评估，不写无损或质保承诺 |
| 灯光合规风险 | 大灯项目可能涉及道路法规和年检边界 | 只写视觉方向，必须增加合规确认提示 |
| 表达过度承诺 | 底盘、平衡杆、轮毂、刹车卡钳等容易被理解为性能承诺 | 只写视觉、保护和适配说明，不写性能结果 |
| 品牌关系误解 | 页面使用理想车型名 | 明确为车型识别，不暗示官方合作 |
