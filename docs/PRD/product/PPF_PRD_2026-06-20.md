### 1.1 目标

为新车车主 / 高端车型车主 / 重视保值率和漆面保护的车主呈现隐形车衣产品页,以「原材料优势 / 制造设备 / 生产工艺 / 技术服务」4 大核心价值背书,展示 8 大系列(貔貅→烛龙)、8 项日常漆面威胁场景、8 项性能等级(★ 0-7)、专车专用施工保障(验收标准),引导到店沟通。

### 1.2 主题色

`orange`(orange-500 主 / orange-400 强调 / orange-950/40 卡片底)。`ProductDetail` 中 PPF 区段背景采用 `bg-black border-y border-zinc-900`(其他产品页为 `bg-zinc-950`),以视觉重量体现「高端防护」定位。

### 1.3 范围

- ✅ **包含**: 隐形车衣详情页的元数据、Hero、tagline 横幅、4 个核心价值、8 项防护场景区、8 大系列参数表、8 项性能等级对比表(★ 0-7)、专车专用施工保障(验收标准)、4 步服务流程、面包屑;与 `/product` 入口的衔接;三视口响应式与 SEO metadata。
- ❌ **不包含**: 单品 SKU 价格表(按系列展示参数,无具体定价);不在本页暗示「原厂漆替代方案」「影响原厂质保」等未经确认措辞;若后续按车型扩展,需独立 PRD 走主题专项模式。

## 4. UI / 交互

### 4.1 视觉规范

- **主色**: `orange-500` / `orange-400` / `orange-950/40`
- **背景**: Hero `bg-zinc-950` + tagline `bg-black border-y border-zinc-900` + 核心价值 `bg-black border-y border-zinc-900` + **防护场景 `bg-zinc-950`** + **系列参数表 `bg-black border-y border-zinc-900`** + **性能对比 `bg-zinc-950`** + **施工保障 `bg-black border-y border-zinc-900`**(因 `serviceSectionBg` 在 PPF slug 下取 `bg-black`,与其他产品页 `bg-zinc-950` 不同)+ 服务流程 `bg-zinc-950`
- **图片容器**: 当前 `productImageMap["ppf"] = "/images/products/ppf.png"`(Hero 区未在 `ProductDetail` 中渲染该图)
- **字体**: Geist Sans
- **圆角**: `rounded-2xl` / `rounded-xl`

### 4.2 组件清单

| 组件 | 路径 | 类型 | 说明 |
| --- | --- | --- | --- |
| `<Header>` | `src/components/Header.tsx` | CC | 全站统一头 |
| `<Footer>` | `src/components/Footer.tsx` | RSC | 全站统一底 |
| `<ProductDetail>` | `src/components/ProductDetail.tsx` | RSC | 6 大产品线共享渲染器;`ppf` 分支渲染 4 个专属区 |
| `Sparkles` / `Check` / `Shield` | `lucide-react` | — | 核心价值 + 防护场景区 icon |
| `<StarRating>` (内置) | `src/components/ProductDetail.tsx` | RSC | 性能等级 ★ 0-7 渲染(7 个 ★ 字符) |

### 4.3 防护场景区(`ppf` 分支)

- **布局**: `grid grid-cols-1 sm:grid-cols-2 gap-4 md:gap-6`
- **卡片内容**: H3 场景名 + 描述
- **数量约束**: 数据驱动 `protectionScenes`(8 项)

### 4.4 系列参数表(`ppf` 分支)

- **布局**: `overflow-x-auto` + `min-w-[900px] text-sm` 横向滚动表
- **列数**: 11 列(系列 / 型号 / 产地 / 材质 / 厚度 / 涂层 / 胶水 / 延伸率 / 耐候性 / 增亮系数 / 质保)
- **表头样式**: `bg-orange-950/40 text-orange-300`
- **数量约束**: 数据驱动 `series`(8 个系列:貔貅/青龙/白虎/朱雀/玄武/麒麟/凤凰/烛龙)

### 4.5 性能等级对比表(`ppf` 分支)

- **布局**: `overflow-x-auto` + `text-sm` 横向滚动表
- **列数**: 6 列(系列 / 型号 / 耐黄变 / 抗划痕 / 环保 / 耐撞)
- **星级渲染**: `<StarRating rating={r.yellowing} max={7}>`(橙色填充 + 锌灰空星)
- **数量约束**: 数据驱动 `performanceRatings`(8 项)

### 4.6 专车专用施工保障(`ppf` 分支)

- **触发条件**: `product.slug === "ppf"`(同时覆盖 `window-film` / `color-film`)
- **背景特化**: `serviceSectionBg = "bg-black border-y border-zinc-900"`(PPF 独有)
- **数据来源**: `src/lib/products.ts` 中 `serviceGuarantee.acceptance`(6 项:折痕 / 划痕 / 起泡 / 尘点 / 胶痕 / 包边)

### 4.7 三视口响应式

| 视口 | 行为 |
| --- | --- |
| Desktop 1440 | 防护场景 2 列 / 系列参数表全宽(可能溢出 → min-w-[900px] 横向滚动) |
| Tablet 768 | 防护场景 1 列 / 参数表横向滚动 |
| Mobile 390 | 防护场景 1 列 / 参数表 + 性能表均横向滚动 |

### 4.8 交互细节

- **面包屑**:`产品中心` 可点击 → `/product`
- **系列参数表**:`min-w-[900px]` 触发横向滚动条;移动端必须能滚
- **性能等级表**:静态,无交互
- **CTA**:本页**不**含私有 CTA
