## 发现的问题

### P1-6:缺少登录失败文案回显

**现状:** 错误密码提交后,页面停留在 `/admin/login` 但**没有显示"用户名或密码错误"等任何错误提示**。

**复现:** 在登录页输入 `admin` / `definitely-wrong-password` → 提交 → 页面停留,但无 toast / inline error / form shake。

**影响:** 用户无法知道是用户名错、密码错、还是系统问题。

**建议:** 在表单下方加 inline error:
```tsx
{error && (
  <p className="text-sm text-red-400 mt-2" role="alert">
    {error}
  </p>
)}
```

**优先级:** P1(影响日常 admin 使用体验)

### P2-3:缺少「忘记密码」入口

**现状:** 内网后台一般不需要,但 seed admin 密码 `admin123` 写在 README 里,生产误用风险存在。

**建议:** 暂不补(后台定位,Seed 脚本可重置);但生产前务必改默认密码。

### P3-2:Logo 渲染位置略偏

**现状:** 桌面端 logo 在卡片内 16px 边距处,视觉上略偏向中央,缺少顶部留白。

**建议:** 增加 logo 上方 `mt-8` padding。
