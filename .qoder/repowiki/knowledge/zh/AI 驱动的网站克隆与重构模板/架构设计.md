- **核心框架**：采用 Next.js 16 (App Router) 结合 React 19 与 TypeScript 严格模式，配置 `output: "standalone"` 以优化 Docker 部署。
- **样式架构**：集成 Tailwind CSS v4 与 shadcn/ui (base-nova 风格)，通过 `components.json` 定义路径别名（如 `@/components`, `@/lib`）及设计令牌系统。
- **质量保障**：使用 ESLint 9 配合 `eslint-config-next` 进行代码规范检查，并通过 `tsconfig.json` 启用增量编译与路径映射。
- **容器化设计**：提供多阶段 Dockerfile 实现依赖安装、构建与运行环境的隔离，并通过 docker-compose.yml 区分生产与开发服务。
